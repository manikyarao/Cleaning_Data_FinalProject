Created a script called runanalysis.R which will merge the test and training sets together. 

Prerequisites:
UCI HAR Dataset must be extracted and..
UCI HAR Dataset must be availble in a directory called "UCI HAR Dataset"

Steps:
1. Merging testing and training data sets.
2. Adding labels to the  mean and standard deviation columns
3. Generating a tidy data set containing the means of all the columns per test subject and per activity. 
4. Generating tidy dataset with tab-delimited file called tidy.txt
5. Updated the txt file to reposiroty
